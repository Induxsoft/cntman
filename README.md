# Cntman(Contenido público).
El paquete Cntman (Contenido público) de V12 es el módulo que permite la administración de contenido digital abierto al público, ofreciendo herramientas para gestionar temas, artículos, preguntas frecuentes, foros, blogs, cursos en línea y rutas de aprendizaje. Su propósito es brindar a las organizaciones un espacio centralizado para difundir información, capacitar usuarios y fomentar la interacción en plataformas web integradas con el ERP.
